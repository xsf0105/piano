# Develop & Installation

<!-- <img src="./public/demo.png"> -->
开发，构建和运行。

```bash
# 获取远程仓库代码
git clone https://github.com/xsf0105/piano

# 进入目录
cd piano

# 安装依赖
npm install

# 启动项目
npm start

# 在浏览器访问 http://localhost:3000
```
